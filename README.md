# Maintenance Tracker

Angular frontend for the Maintenance Tracker application. Authenticates via Azure AD (MSAL) and connects to the Maintenance Tracker API.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

## Build

Run `ng build` to build the project (defaults to development configuration). The build artifacts will be stored in `dist/maintenance-ui/`.

Run `ng build --configuration=production` for a production build.

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI overview](https://angular.dev/tools/cli).
